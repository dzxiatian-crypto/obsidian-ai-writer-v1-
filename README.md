# AI Writer - Obsidian 插件

## 功能

- ✨ **AI 续写** - 选中文字，一键续写
- ✨ **AI 润色** - 选中文字，AI 帮你改得更好
- ✨ **AI 总结** - 选中文字，自动生成总结

## 安装

1. 打开 Obsidian
2. 进入设置 → 第三方插件
3. 关闭安全模式
4. 将文件夹复制到：`vault/.obsidian/plugins/`
5. 重新启动 Obsidian
6. 启用插件

## 使用

1. 选中一段文字
2. 按 `Ctrl+P` 打开命令面板
3. 输入 `AI Continue` 或 `AI Polish` 或 `AI Summarize`
4. 等待 AI 生成
